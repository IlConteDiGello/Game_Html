Avventura grafica, almeno un bottone ogni stanza, verranno assegnati obbiettivi si potrà scegliere se seguirli o meno.

piccola idea: colore i bottoni in base alle azioni, se azione va contro l'obbiettivo il bottone sarà per esempio rosso

scatola dei cereali (percorso 1)
Dio(percorso 2)
specchio (combattimento extra, contenuto extra)

finire il diagramma delle stanze e della storia

mettere le transizioni da stanza a stanza
